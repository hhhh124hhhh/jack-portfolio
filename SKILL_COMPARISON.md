# 单个提示词 vs 多提示词 Skill 对比

## 📦 创建的两个 Skill

### 1. chatgpt-prompts-skill（多提示词合集）
- **大小**: 16KB
- **内容**: 50+ 提示词（开发、翻译、创意、职业、教育等）
- **文件**: SKILL.md + references/prompts.md (42KB)
- **用途**: 全能型助手，覆盖多场景

### 2. job-interviewer-skill（单功能专注）
- **大小**: 4.0KB
- **内容**: 1 个核心功能（求职面试）+ 扩展资料
- **文件**: SKILL.md + references/interview-questions.md (8KB)
- **用途**: 专注于求职面试准备

---

## ⚖️ 详细对比

| 维度 | 多提示词 Skill | 单提示词 Skill |
|------|---------------|---------------|
| **文件大小** | 16KB | 4.0KB（75% 更小） |
| **提示词数量** | 50+ | 1 个核心 + 扩展 |
| **专注度** | 广泛覆盖 | 深度聚焦 |
| **加载速度** | 相对较慢（需要搜索）| 极快（直接加载） |
| **维护成本** | 高（需持续更新）| 低（单一功能） |
| **定位** | 通用工具 | 专家工具 |
| **目标用户** | 多场景用户 | 特定需求用户 |

## 🎯 适用场景

### 多提示词 Skill 适合：
1. **多场景用户** - 需要多种AI能力
2. **内容创作者** - 需要灵感和不同风格
3. **学生和自学者** - 需要多工具辅助学习
4. **预算有限但需求多样** - 一次购买满足多需求

### 单提示词 Skill 适合：
1. **有明确需求** - 明确知道要面试、要创作等
2. **专业人士** - 需要深度专业帮助
3. **短期使用** - 特定时期（求职季、项目期）
4. **首次体验** - 价格门槛低，愿意尝试
