## 学习：Clawdbot 命令和目录结构

### 重要发现
- Clawdbot 数据目录在 `~/.clawdbot`，**不是** `/var/lib/clawdbot`
- 用户之前在错误的目录搜索导致失败

### Clawdbot 目录结构
```
~/.clawdbot/
├── agents/main/sessions/     # 会话日志
├── cron/                     # 定时任务
├── credentials/              # 凭证
├── devices/                  # 设备配置
├── extensions/               # 扩展插件
├── media/                    # 媒体文件
├── skills/                   # 技能
├── subagents/                # 子代理
├── identity/                 # 身份配置
├── secrets/                  # 密钥
├── clawdbot.json            # 主配置
└── .env                     # 环境变量
```
