项目目标

自动化发现互联网上的优质 AI 提示词，使用多维度评估系统筛选出最有价值的内容，然后将其转换为可直接使用的 Clawdbot Skills。这不仅让提示词变得可复用，还建立了 AI 技能的商业化生态。

  • ✓ 多源数据收集（Twitter/X、Reddit、GitHub、Hacker News）
  • ✓ 语义去重算法，避免重复内容
  • ✓ LLM 驱动的质量评估系统
  • ✓ 自动转换为 Clawdbot Skills
  • ✓ 自动发布到 ClawdHub

自动化工作流

1

数据收集

从多个平台抓取 AI 提示词相关内容，包括推文、Reddit 帖子、GitHub 讨论、Hacker News 等。

Twitter API Reddit API GitHub API SearXNG
2

语义去重

使用 sentence-transformers 模型计算提示词之间的语义相似度，自动过滤重复和高度相似的内容。

PyTorch sentence-transformers all-MiniLM-L6-v2 相似度阈值: 0.85
3

质量评估

使用 Claude (coding-agent) 从创新性、实用性、清晰度、可复用性四个维度评估提示词质量。

Claude LLM Judge 4 维度评分 100 分制
4

Skill 转换

将高分提示词自动转换为标准的 Clawdbot Skill 格式，生成 SKILL.md、示例代码和使用文档。

自动模板生成 SKILL.md README.md .skill 打包
5

自动发布

将生成的 Skills 自动提交到私有仓库，并通过 ClawdHub CLI 发布到技能市场。

Git 自动提交 ClawdHub API 自动定价 市场发布
