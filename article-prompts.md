## 📖 使用 ai-music-prompts skill

### 什么是 ai-music-prompts skill？

这是一个专门为 AI 音乐生成优化的技能（skill），包含：
- ✅ 丰富的提示词模板（流行、古风、说唱等）
- ✅ Mureka V8 专属优化
- ✅ 中英双语支持
- ✅ 中文音乐生成最佳实践

### 如何调用 skill

在 Claude Code 中直接说：

```
/skill ai-music-prompts
```

或者直接提问：

```
帮我创建一个华语流行情歌的提示词
```

```
用 Mureka V8 生成一首古风歌曲
```

### 与 skill 对话示例

**示例1：创建情歌提示词**
```
你: 帮我写一个华语流行情歌的提示词
AI: 好的！这里是一个专为 Mureka V8 优化的提示词：
    "华语流行 with 温暖女声, 深情, 钢琴弦乐, 90 BPM - 关于思念的情歌"
```

**示例2：创建古风提示词**
```
你: 我想要一首仙侠风格的古风歌曲
AI: 明白！试试这个提示词：
    "古风 with 空灵女声, 悠扬清幽, 古筝笛子, 75 BPM - 仙侠风格，清幽脱俗"
```

## 🎯 5个精选测试提示词

> 所有提示词均来自 ai-music-prompts skill，专为小白用户设计，简单易用！

### 1. 华语流行情歌 ⭐（最简单）

**提示词**：
```
华语流行 with 温暖女声, 深情, 钢琴弦乐, 90 BPM - 关于思念和爱情的感人情歌
```

**英文版**：
```
Mandopop with warm female vocals, emotional, piano and strings, 90 BPM - A heartfelt love song
```

**应用场景**：
- 💕 表白、纪念日、情人节
- 🎬 情感类短视频、Vlog
- 🎵 送给恋人、朋友的专属歌曲

**难度**：⭐（极简，完全不需要专业知识）

**预期效果**：
柔和感人的抒情歌，温暖女声，钢琴和弦乐伴奏，适合表达思念和爱意。

**💡 小贴士**：
- 可以把"女声"改成"男声"
- 可以把"深情"改成"欢快"，就变成甜蜜情歌
- 完全不需要懂乐理，复制粘贴就能用！

---

### 2. 古风仙侠 ⭐⭐（中国特色）

**提示词**：
```
古风 with 空灵女声, 悠扬清幽, 古筝笛子, 75 BPM - 仙侠风格，清幽脱俗
```

**英文版**：
```
Gufeng with ethereal female vocals, melodious and serene, guzheng and dizi, 75 BPM - Xianxia style
```

**应用场景**：
- ⚔️ 游戏配乐、仙侠视频
- 🏮 古风短视频、汉服摄影
- 📖 朗读古诗词背景音乐

**难度**：⭐⭐（稍微需要一点东方美学）

**预期效果**：
仙气飘飘的东方音乐，古筝和笛子主奏，空灵女声，如仙境般美丽。

**💡 小贴士**：
- 可以改成"武侠风格"变成豪迈的古风
- 适合配合古风视频、汉服展示
- Mureka V8 的古风非常出色，强烈推荐！

## 🚀 使用流程（小白三步曲）

### Step 1: 调用 skill
```
输入：/skill ai-music-prompts
或者：帮我创建一个XX风格的提示词
```

### Step 2: 复制提示词
从 skill 返回的结果中，复制提示词（可以稍作调整）

### Step 3: 粘贴到 Mureka V8
打开 Mureka V8，粘贴提示词，生成音乐！

---

## 💡 进阶技巧

### Level 1: 复制粘贴（⭐）
直接使用上面5个提示词，100%可用！

### Level 2: 修改情绪（⭐⭐）
- 把"深情"改成"欢快"
- 把"温暖女声"改成"激昂男声"
- 调整 BPM 数值

### Level 3: 组合元素（⭐⭐⭐）
- 参考多个提示词，组合元素
- 添加自己的创意描述
- 描述具体的使用场景
