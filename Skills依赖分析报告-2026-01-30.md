## 🎯 可直接使用的 Skills 详情

### 1️⃣ 知识型 Skills（纯内容，零依赖）

这些 skills 提供高质量的知识库和模板，无需任何配置：

#### 📚 chatgpt-prompts
- **内容**: 143k+ GitHub 项目的精选提示词
- **功能**: 角色扮演、专业任务、创意写作等
- **使用**: 直接调用，无需配置
- **价值**: ⭐⭐⭐⭐⭐ (社区验证)

#### 🎵 ai-music-prompts
- **内容**: AI 音乐生成提示词
- **平台**: Suno、Udio、Mureka V8
- **特色**: 双语支持（中英文），优化 Mureka 中文生成
- **使用**: 查询模板即可使用

#### 📖 prompt-learning-assistant
- **内容**: 58+ 种提示词技术系统化学习
- **结构**: 入门/进阶/专家分级
- **功能**: 知识库、实战案例、学习路径
- **使用**: 直接学习，无需外部服务

#### 🔧 prompt-optimizer
- **内容**: 提示词优化（58 种技术）
- **功能**: 评估、优化、增强提示词
- **使用**: 输入提示词，获得优化建议

### 2️⃣ 工具型 Skills（本地功能）

这些 skills 使用本地工具或 Clawdbot 内置能力：

#### 💼 job-interviewer
- **类型**: 交互式模拟
- **功能**: 模拟任何职位的面试官
- **使用**: 对话即可，无需 API
- **适配**: 所有职位类型

#### 📄 resume-builder
- **类型**: 文档生成
- **功能**: 简历、求职信、LinkedIn 简介
- **特色**: ATS 友好格式
- **使用**: 输入信息，生成文档

#### 📅 calendar
- **类型**: 系统集成
- **功能**: 创建事件、管理会议
- **依赖**: `curl`、`jq`（标准命令行工具）
- **使用**: 本地命令行调用

#### 🔒 clawdbot-security-check
- **类型**: 安全审计
- **功能**: Clawdbot 配置安全检查
- **权限**: 只读文件访问
- **使用**: 内部能力，无需外部服务

#### 📈 x-trends
- **类型**: 数据获取
- **功能**: X/Twitter 实时热门话题
- **使用**: 调用即可，无需 API key

### 2️⃣ tiktok-ai-model-generator

**类型**: 工作流指导（非直接依赖）  
**第三方工具**: Pinterest、Claude、Nano Banana Pro、Veo/Kling

**4 步工作流**:
1. ✅ **Pinterest**: 手动选择参考图（免费）
2. ✅ **Claude**: 生成 JSON 提示词（已有）
3. ⚠️ **Nano Banana Pro**: 图像生成（需账号）
4. ⚠️ **Veo/Kling**: 视频动画（需账号）

**特点**:
- Skill 本身是**知识指导**
- 教你如何组合多个 AI 工具
- 可选择使用哪些工具

**使用成本**:
- Pinterest: 免费
- Claude: 已在使用
- Nano Banana Pro: 需注册（可能有费用）
- Veo/Kling: 需注册（可能有费用）

**价值**: ⭐⭐⭐⭐ (电商卖家、内容创作者)

**灵活性**: 可只用前 2 步（免费部分）
