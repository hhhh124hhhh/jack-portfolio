# Achievement System - 个人成长追踪工具 🏆

一个深色科技风格的命令行成就系统，帮助你追踪和达成个人目标！

## ✨ 特性

- 🎨 **深色科技风格** - 使用 Rich 库打造的精美终端界面
- 🎯 **成就系统** - 创建自定义成就，追踪进度
- 📊 **统计信息** - 实时查看完成率和积分
- 🎉 **解锁庆祝** - 达成成就时的精美庆祝面板
- 💾 **本地存储** - JSON 数据存储，无需数据库
- 🔧 **易于扩展** - 模块化设计，方便添加新功能

## 🚀 快速开始

### 安装依赖

```bash
pip install click rich
```

### 初始化系统

```bash
./ach init
```

### 基本命令

```bash
# 查看所有成就
./ach list

# 查看当前进度
./ach status

# 查看统计信息
./ach stats

# 添加进度
./ach add <achievement_id> <amount>

# 解锁成就
./ach unlock <achievement_id>
```
